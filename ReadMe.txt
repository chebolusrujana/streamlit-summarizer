# 📄 AI Document Summarizer with ROUGE Evaluation

This Streamlit app uses **Facebook BART-Large-CNN** to summarize text, PDF, and Word documents.  
It also computes **ROUGE scores** if you provide a reference summary.

---

## ✨ Features
- 📂 Upload `.txt`, `.pdf`, or `.docx` files
- 📝 Summarizes documents into concise text
- 📊 Computes **ROUGE-1, ROUGE-2, and ROUGE-L** scores if a reference summary is given
- 📥 Download the generated summary
- 🎨 Optional background image support
